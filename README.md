Student Spending Analysis

Project description

This project focuses on analyzing student spending behavior based on the Student Spending Dataset.
The goal is to understand how students distribute their expenses across different categories.

Objectives

Analyze overall student spending
Identify top spending categories
Compare spending behavior by gender
Calculate key metrics (total, average, min, max)
Present insights through a simple dashboard

Data source

Dataset: Student Spending Dataset
Platform: Kaggle
Tools & Skills

Key insights

Tuition dominates total spending, accounting for the majority of expenses
Spending significantly exceeds available funds, indicating a large financial deficit
Total spending is more than 4 times higher than income
Spending patterns are consistent across genders, with differences only in total amounts
Average spending remains stable across academic years

Conclusion

The analysis shows that student spending is heavily concentrated in essential categories, particularly tuition. At the same time, expenses significantly exceed available funds, highlighting a potential financial imbalance. Spending behavior is generally consistent across different groups, suggesting similar financial priorities among students.
